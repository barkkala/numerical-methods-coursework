# Курсовая работа по численным методам

Это моя курсовая работа по численным методам. Программа написана на Python, интерфейс сделан через Tkinter.  
В одном окне собраны все части работы: интегралы, нелинейные уравнения, интерполяция, МНК и метод конечных разностей.

## Что есть в проекте

### 1. Интегралы

В первой части считаются два определённых интеграла:

- f1(x) = sqrt(0.7x^2 + 2.3) / (3.2 + sqrt(6.8x - 1.4))
- f2(x) = cos(0.4x + 0.6) / (0.8 + sin(x + 0.5)^2)

Отрезки по умолчанию:

- для f1: [0.5; 1.8]
- для f2: [0.4; 1.2]

Используются методы левых, правых и средних прямоугольников, метод трапеций, метод Симпсона и уточнение по правилу Рунге.

### 2. Нелинейные уравнения

Во второй части можно выбрать одно из уравнений:

- 3x^4 + 8x^3 + 6x^2 - 10 = 0
- 3^x - 2x - 5 = 0
- 2x^2 - 0.5^x - 3 = 0
- x * log(x + 1) = 1

Для поиска корней добавлены методы дихотомии, Ньютона, хорд, комбинированный метод и метод простых итераций. Также есть график функции и автоподбор интервала.

### 3. Интерполяция

Третья часть строит интерполяционный полином по таблице значений.

Исходные точки:

- x: -0.2, 0.0, 0.2, 0.4, 0.6, 0.8
- y: 0.0, -0.2, 0.4, 2.0, 4.0, 4.8

По умолчанию значение считается в точке x* = 0.1.

В программе есть полином Лагранжа, полином Ньютона и канонический вид. Для канонического вида можно выбрать решение СЛАУ методом Гаусса или Крамера.

### 4. Метод наименьших квадратов

Четвёртая часть сделана для варианта 14. В ней используются такие исходные данные:

- x: 1.01, 1.02, 1.12, 1.23, 1.34, 1.46, 1.55, 1.60, 1.77, 1.87, 1.95, 2.00
- y: 1.20, 3.20, 4.19, 4.39, 4.58, 5.73, 6.83, 5.93, 5.51, 4.81, 2.94, 1.12

Проверяются три модели:

- y = ax + b
- y = ax^2 + bx + c
- y = c + b/x + a/x^3

Для каждой модели считаются коэффициенты и сумма квадратов отклонений `S`. Системы можно решать методом Гаусса, Крамера или через обратную матрицу.

### 5. Метод конечных разностей

Пятая часть решает краевую задачу:

y'' - (x / (x + 1))y' + 2xy = ln(x)

Граничные условия:

- y(1.0) = 0.2
- y(2.5) = 1.0

По умолчанию берётся n = 7.

Есть метод прогонки, метод Гаусса, метод Крамера и решение через обратную матрицу. После расчёта строится график решения и интерполяционный полином по найденным точкам.
